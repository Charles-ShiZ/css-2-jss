# CSS to JSS

这是一个帮助你将`CSS转化为JSS`或者将`JSS转化为CSS`的vscode插件

## Usage

## Editer Context
* `CSStoJSS`
* `JSStoCSS`

### Keybindings
* `ctrl+q` 将CSS转化为JSS
* `ctrl+e` 将JSS转化为CSS
